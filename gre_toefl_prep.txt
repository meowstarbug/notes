单词：
    背单词是必须的

    不用背完再做题，稍微有一个小一点的pool之后，就可以一边学新词，一边做easy的单词练习应试技巧了
    优质练习题 很适合检查 / 扩充词汇量
    不过认识率在70%以下的话，建议先背单词，不要浪费了题目

    大部分时候并不是考准确的翻译，而是大致意思，但对“有几种释义（也就是一词多义的释义数目）”，以及“释义的衍生用法”，还有同义词反义词要求较高
    综上建议用webster + thesaurus背单词，远离天朝各大翻译网站，中文释义不仅很多时候覆盖有限，还在解题时体验极差（）

    背单词策略：
        佛脚 / 乐词GRE精选词汇（其实不确定这两个关系，但应该是差不多的）超好用
        远离6000词的那个pool，不论它叫什么（红宝书 / 绿宝书 / GRE全部词汇...），连新东方老师自己都不屑于用那玩意
        媒介看个人偏好，我用的乐词，也有人用的纸书...
    
    一些媒介测评：
        乐词的艾宾浩斯，按多久背完规划工作量，以及多种形式选择题，甚至还有一些优秀的错题陷阱，我吹爆，真实吹爆，一个月背完全靠它（不过背自己熟悉不少词的pool建议绕开一切一个个词看的app...）
        扇贝 / Anki / 其它flashcards风格应用，很火，攻略也多，但我跟我认识的另一个大佬都先后弃坑过这种背法...主要问题是，自己模糊地记得单词的意思时，很难得到一个确认，主观感受并不好
        百词斩也是选择题，不过据悉，很多人被里面的图形选择题祸害，不背释义只背图（）
        纸质书背会的词很多的pool很好用（比如我的托福），但一页页纸全是新词的时候，很多背法都会指数型变痛苦...
        tldr 我爱乐词收钱也爱

填空：

    先复制粘贴一些背单词时提到的：
        不用背完再做题，稍微有一个小一点的pool之后，就可以一边学新词，一边做easy的单词练习应试技巧了
        优质练习题 很适合检查 / 扩充词汇量
        不过认识率在70%以下的话，建议先背单词，不要浪费了题目

        大部分时候并不是考准确的翻译，而是大致意思，但对“有几种释义（也就是一词多义的释义数目）”，以及“释义的衍生用法”，还有同义词反义词要求较高
    
    填空题本质上都是一毛一样的东西
    都是单词题
    需要的技能，都是：词汇量 + 少量词根词缀词源知识 + 在原文中找到空位线索
    选项数以及选几个，主要影响题目的恶心程度以及有多容易翻车

    大概解题方法：找线索确认空里大概应该是什么意思的单词 - 在选项中划掉几个显然不是的 - 再挑一个最像的
    princeton里详细讲了这个策略

    如果在单词含义没问题的前提下，不知道为什么是这一项，找不到supporting evidence
    建议还是补充一些理论知识（虽然理论上princeton够了）
    
    如果对单词含义的自然延伸不了解，经常盯着单词，知道它自己的意思，但不理解这个语义，不知道为什么是正确答案
    建议补点关于词根词缀，以及词义延伸，修辞手法的知识
    这个是中英表达差异的锅，知乎GRE攻略里应该有一些会提到，princeton默认你native所以这个问题完全没讲

阅读：
    技能点：
        逻辑单题：一个一道题的迷你阅读，主要是看懂题+推理，算个技能点吧
        长难句：似乎是很多人爱提到的一个知识点；我学了一个拆解法之后就随便过了，感觉不难（我可以把那个法子大概教你）
        文章结构拆解：也被称为“如何快速看完原文”，速度的基本保证（四舍五入是万物之本，阅读超时很常见，后果很惨）

        以上技能建议有几本掌握之后再开始做题
        填空如果你看不懂单词，你还可以换easy的题目先练，难题先照着选项背单词；
        阅读要是基础技能不熟，长难句看不懂，就要烧时间啃，然后结构拆不好也会耗时间，然后你就很容易超时，还做得一般...
        要是故意多花时间做的话，又完全练不到策略了，因为阅读磨时间总是能做完的，只是按时做完做对需要各种技巧...
    
    阅读题分好几种，除掉最特别的逻辑单题，其它的印象里还是有一定的相通性的？
    阅读本质上都是理解原文之后，按题目要求找到信息点并parse成每一个选项是对是错
    阅读原文的本质是（个人观点）大型撕逼吃瓜会（）
    所有的阅读题都对单词没有要求，你看到的所有单词，要么是托福 / 六级水平，要么是“你知道这是个专业名词就好了，它是啥不重要”
    总之如果词汇量影响了你做题，这很有可能是托福 / 雅思备考要解决的部分...
    需要的技能点除了上面写的，就是overall的理解能力了
    princeton的阅读讲的也不错

    阅读建议时刻计时做，理由见“以上技能建议有几本掌握之后再开始做题”，掌握做题技巧之后，一开始给自己放水一些时间也可以，但绝对不能毫无策略地一篇篇磨过去

    自己在用的一些策略：
        长难句用的某个xdf公开课上讲的表示法，大概如下：
        将每一个单句分解成：
        主语→（谓语）→宾语（谓语在箭头上）
        然后定状补全部写在上述主干词汇的下面
        不同的单句之间（或者有大从句的话），用额外的箭头描述逻辑关系
        一个长难句，一般四个谓语，不能再多了

        快速啃文章：将整片文章看作一个（微博极其常见的）撕逼瓜的总结，然后用吃瓜的思路跟阅读风格理解文章
        GRE阅读经过你的解析应该大概长这样：
        首先，我们在讨论这个主题，x（瓜的主角）
        第一位撕逼选手a，提出了论据（吃瓜中的“锤”）a1 a2 a3，他的观点是a0
        第二位撕逼选手b，提出了b1 b2，反对a0，认为b0
        第三位撕逼选手c，觉得a0 b0怎么怎么样，提出c0，以及论据c1 c2
        以此类推
        有时候撕逼选手里会混入作者，问题不大，作者不会把自己观点渗透对别人的观点的描述的，基本一定是自己单独一大段，作者有没有加入撕逼大战，都不影响吃瓜方法


作文：
    Issue：
        时间跟字数推荐我忘了
        本质是让你选个边，然后按照题目描述写一组x段论
        就，首先它会有一段（什么英语考试写作都有的）主题描述
        然后底下会有一个写作要求，就是告诉你在写作时要侧重哪方面的说明
        一定程度上（大部分人类的能力范围内），issue越长越好

        我一般的写法（也是厨师班统一教的写法）如下：
            开头结尾，中间三段
            观点上选一边，不要折衷
            先准备论据，看下站哪边比较好写
            然后排一下观点顺序，看下哪段写让步
            然后中间的三段的前两段中，挑一段写，自己反方的观点为什么不对
            也就是一个“反正正”或者“正反正”的总结构
            每一段提供一到两个证据支持
            在进行证明时注意按照要求来进行展开，千万别学托福肆意生长，一共不到十种要求，背板就好

    Argument：
        本质是给你一个亲自下场怼人的机会
        题目会塞给你一段漏洞百出的短文，以及画风跟楼上issue差不多的一句写作要求
        argument字数显然也是越多越好，但相比issue内部结构和逻辑性会重要性也上升一些

        我一般的写法（也是厨师班统一教的写法）如下：
            先看文章，列出文章的逻辑链，以及里面有什么漏洞
            挑出里面最好怼的3-4个漏洞（有那种可以骂很多的大漏洞的话，两个也行
            三段式，每段按描述进行展开

    有本书叫新GRE5.5，两科基本都有保姆式教学
    不过印象里里面讲的套路甚至有些过多了，用自己最习惯的就好（或者找个保姆课 / 大佬，要一个套路，学完就用到天荒地老哈哈哈哈）


托福相关


GRE跟托福之间的关系？


一些有的没的乱七八糟的鬼点子

关于错题：
    我为什么错了
        我为什么没有选正确的那一项
        我为什么选了错的那项
        是什么迷惑到我，让我觉得正确的那项不对
        是什么迷惑到我，让我觉得错误的那项是对的
        我能被迷惑是因为缺乏了什么知识 / 技能
    其余的选项
        它们被我划掉时，我找的原因是对的吗？
        每一项对 / 错的原因都要找清楚，单词题弄懂每个释义，以及相关的同义词反义词，阅读题找到证明 / 证伪它的信息点
        

关于模板：
    你对这个模板的理解程度，使用的灵活度，使用的顺手度，绝大部分时候，会远比模板本身的质量要重要
    模板可以指：词汇 / 句式 / 文章结构，通用
    主观题通用
    
    由此衍生出的各种关于模板的说法
        （哪都有）自己做的模板永远是最稳的
        （托福口语）塞进手里的模板能背熟也不是不可以，但要是这个模板对你来说太难了，建议降低模板难度
        （托福口语）这个单词你要能自然而然地说出来倒是可以，但要是用了这个单词反而让你说的磕磕绊绊的，或者你就那几个单词硬塞，不如不用
        （托福作文）死背成品开头结尾，要是被查重会死的很惨，就算绕过了查重，用得不好硬嵌，依然会死的很惨...
        （托福口语）你要是有那个本事，自然的表达就是那个水平的话，那真的完全不需要模板
        （GRE作文）比较完整的策略一般是，先确认自己适合什么难度的模板，然后观察一些这样的模板后，再制作自己用的顺的模板
        （接上）自己这样捏出的模板，一般会比较好扩展（加大复杂度），或者是变通到不同场景跟题型

关于对题目的了解：
    这题有多少总时间？
    有多少时间是实际作答，有多少时间要做别的？
    这题的答案大概的类型？（客观 / 主观）
    
    客观组：
        答案在出题时会大概被怎么设置？
        相应地，这道题的答案应该怎么获得？
        有可能怎么错掉？
        一问大概有多少时间来找到答案？
    
    主观组：
        答案的结构，应该大概长什么样？
        在这个架构下，每一个环节大概有什么内容？
        应该如何提取信息来织出自己要的答案的样子？
        如果自己的作答不尽人意，可能是因为什么？怎么改良？
